# Recommendation Model Prototype

## 개요

이 폴더는 RestArt 추천 모델 프로토타입에서 사용하는 이미지 분석 유틸리티 코드를 정리한 공간이다.

RestArt의 추천 기능은 사용자가 업로드한 공간 이미지와 작품 DB의 색상·감성 데이터를 비교하여 어울리는 작품을 추천하는 구조이다.

---

## 파일 구성

| 파일             | 역할                                                                            |
| ---------------- | ------------------------------------------------------------------------------- |
| `image_utils.py` | 이미지 로딩, 유사도 비교, 대표 이미지 선정, 색상 분석에 사용하는 유틸 함수 모음 |

---

## 주요 기능

`image_utils.py`는 다음 기능과 연결된다.

### 1. 이미지 URL 처리

이미지 URL을 기반으로 이미지를 불러오고, 추천 또는 리플릿 생성에 사용할 수 있는 형태로 변환한다.

### 2. 이미지 유사도 비교

사용자가 촬영한 이미지와 작품 이미지 간의 유사도를 비교한다.

이 기능은 전시장에서 촬영한 이미지가 어떤 작품과 가까운지 판단하는 데 사용될 수 있다.

### 3. 대표 이미지 선정

여러 이미지 중 리플릿에 사용할 대표 이미지를 선정한다.

이 과정은 개인화 모바일 리플릿 생성 기능과 연결된다.

### 4. 색상 기반 분석

이미지에서 색상 정보를 추출하고, 추천 모델의 입력값으로 사용할 수 있도록 정리한다.

---

## RestArt 추천 흐름과의 관계

추천 흐름은 다음과 같다.

```text
사용자 이미지 업로드
→ 이미지 색상 분석
→ 작품 DB와 색상·감성 정보 비교
→ 추천 작품 후보 생성
→ 추천 결과 반환
```
