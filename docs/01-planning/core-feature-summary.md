# Core Feature Summary

## 개요

RestArt의 핵심 기능은 사용자의 공간과 취향을 분석하여 예술 작품을 추천하고, AR 시뮬레이션과 거래 기능으로 연결하는 것이다.

서비스는 크게 다음 기능 그룹으로 구성된다.

1. AI 작품 추천
2. AR 작품 설치 시뮬레이션
3. 작품 거래 및 컬렉션
4. 개인화 모바일 리플릿
5. 작가 및 전시 연계 기능

## 1. 공간 이미지 업로드

사용자는 작품을 설치하고 싶은 공간의 사진을 업로드한다.

업로드된 이미지는 작품 추천을 위한 기본 입력값으로 사용된다. 서비스는 이미지에서 주요 색상과 색상 비율을 추출하고, 이를 추천 알고리즘의 입력 데이터로 변환한다.

### 주요 기능

- 공간 사진 업로드
- 이미지 품질 확인
- 추천 정확도를 위한 촬영 가이드 제공
- 이미지 기반 색상 추출

## 2. 색상 추출

업로드된 공간 이미지 또는 작품 이미지에서 대표 색상을 추출한다.

대표 색상은 단순 RGB 값만 저장하는 것이 아니라, 색상 비율과 함께 저장된다. 이를 통해 작품과 공간의 색상 구성을 비교할 수 있다.

### 주요 기능

- k-means clustering 기반 대표 색상 추출
- RGB 값 저장
- 색상 비율 저장
- 작품 DB와 추천 알고리즘에 활용

## 3. 감성 분석

RestArt는 색상과 작품 설명을 감성 데이터로 변환하여 추천에 활용한다.

작품 이미지, 작품 제목, 작품 설명, 생성된 텍스트 데이터를 함께 활용하여 작품의 감성을 추출한다. 추출된 감성은 RestArt의 감성 테이블에 매핑되어 추천 정확도를 높이는 데 사용된다.

### 주요 기능

- 멀티모달 기반 작품 감성 분석
- 작품 설명 텍스트 생성
- 감성 카테고리 매핑
- 작가 감성 수정 기능
- 수정된 감성 데이터를 재학습 또는 개선 데이터로 활용

## 4. AI 작품 추천

사용자의 공간 색상과 감성 데이터를 기반으로 작품을 추천한다.

추천 과정은 색상 유사도와 감성 일치도를 함께 고려한다. 감성 일치 개수를 우선하고, 동일한 조건에서는 켄달타우 기반 점수를 활용하여 정렬한다.

### 추천 기준

1. 감성 일치 개수
2. 색상 및 비율 유사도
3. 켄달타우 점수
4. 작품 다양성
5. 사용자 편향 최소화

## 5. AR 작품 설치 시뮬레이션

추천된 작품을 실제 공간에 가상으로 배치한다.

사용자는 작품을 구매하기 전에 자신의 공간에 작품이 어울리는지 확인할 수 있다.

### 주요 기능

- 작품 이미지 선택
- 공간 내 작품 배치
- 크기 및 위치 확인
- 구매 전 시각적 검증
- 모바일 기반 AR 경험 제공

## 6. 작품 상세 조회

사용자는 추천된 작품의 상세 정보를 확인할 수 있다.

### 제공 정보

- 작품 이미지
- 작품명
- 작가명
- 작품 설명
- 감성 태그
- 대표 색상
- 가격
- 작품 거래 정보
- 작가 페이지 연결

## 7. 작품 거래 및 리세일

RestArt는 작품 구매와 리세일 구조를 포함한다.

사용자는 작품을 구매하고 자신의 컬렉션에 저장할 수 있으며, 이후 리세일 기능을 통해 작품을 재거래할 수 있다.

### 주요 기능

- 작품 구매
- 작품 보유 목록 관리
- 컬렉션 공개
- 리세일 등록
- 거래 기록 관리

## 8. 작가 페이지

작가는 자신의 작품과 활동 정보를 플랫폼에 등록할 수 있다.

### 주요 기능

- 작가 프로필
- 작품 목록
- 작가 소개
- 협업 및 전시 정보
- 팔로우 또는 관심 작가 기능

## 9. 사용자 컬렉션

사용자는 구매하거나 관심 있는 작품을 컬렉션으로 관리할 수 있다.

### 주요 기능

- 관심 작품 저장
- 구매 작품 관리
- 컬렉션 공개
- 취향 기반 추천 개선
- SNS 공유

## 10. 개인화 모바일 리플릿

전시 관람객이 촬영한 작품을 기반으로 개인화된 모바일 리플릿을 생성한다.

### 리플릿 구성

- 전시 정보
- 사용자가 선택한 대표 사진
- 촬영 작품 중 대표 작품
- 취향 분석 결과
- 추천 작품
- 추천 전시
- SNS 공유 기능

## 11. 전시 경험 기반 추천

사용자가 전시장에서 촬영한 작품 이미지를 분석하여 취향을 도출하고, 비슷한 작품이나 전시를 추천한다.

### 주요 기능

- 촬영 이미지 저장
- 작품 이미지 분석
- 색상·감성 기반 취향 분석
- 추천 작품 생성
- 추천 전시 연결

## 12. SNS 공유

사용자는 자신의 리플릿, 컬렉션, 추천 작품을 SNS로 공유할 수 있다.

이 기능은 개인의 기록을 넘어 전시와 작가의 홍보 효과로 확장될 수 있다.
