# User Scenario

## 개요

RestArt는 사용자, 예술인, 전시 운영자 각각의 문제를 해결하기 위해 설계되었다.

이 문서는 주요 사용자 유형별 사용 시나리오를 정리한다.

---

## Scenario 1. 집에 어울리는 작품을 찾고 싶은 사용자

### 사용자 상황

사용자는 새로 꾸민 거실이나 방에 어울리는 작품을 찾고 싶지만 어떤 작품이 공간에 어울릴지 판단하기 어렵다.

온라인에서 작품 이미지를 보더라도 실제 공간에 걸었을 때의 크기, 색감, 분위기를 예측하기 어렵다.

### 사용 흐름

1. 사용자는 RestArt에 접속한다.
2. 작품을 설치하고 싶은 공간 사진을 업로드한다.
3. 서비스는 공간 이미지에서 주요 색상과 분위기를 분석한다.
4. AI 추천 시스템은 공간과 어울리는 작품을 추천한다.
5. 사용자는 추천 작품을 AR로 공간에 배치해본다.
6. 작품이 공간에 어울린다고 판단되면 상세 정보를 확인한다.
7. 사용자는 작품을 구매하거나 컬렉션에 저장한다.

### 기대 효과

- 작품 선택 부담 감소
- 구매 전 공간 적합성 확인
- 온라인 작품 구매의 불확실성 감소
- 작품 구매 전환 가능성 증가

---

## Scenario 2. 장애·신진 예술인의 작품을 발견하고 싶은 사용자

### 사용자 상황

사용자는 유명 작가 중심의 미술 시장보다, 신진 작가나 사회적 가치를 가진 예술 소비에 관심이 있다.

하지만 장애·신진 예술인의 작품을 한곳에서 발견하거나 비교하기 어렵다.

### 사용 흐름

1. 사용자는 RestArt에서 추천 작품을 확인한다.
2. 추천 작품의 작가 페이지로 이동한다.
3. 작가 소개와 작품 세계를 확인한다.
4. 관심 작가를 팔로우하거나 작품을 컬렉션에 저장한다.
5. 작품을 구매하거나 이후 전시 정보와 연결된다.

### 기대 효과

- 인지도가 낮은 작가의 작품 발견
- 가치 소비 경험 제공
- 예술인과 사용자 간 연결 강화
- 작품 구매 경험의 의미 확장

---

## Scenario 3. 전시 경험을 기록하고 싶은 관람객

### 사용자 상황

사용자는 전시장에서 본 작품을 사진으로 남기지만, 관람 후에는 사진만 남고 전시 경험이 정리되지 않는다.

기존 리플릿은 모든 관람객에게 동일한 정보를 제공하기 때문에 개인의 취향이나 관람 경험을 충분히 반영하지 못한다.

### 사용 흐름

1. 사용자는 전시장에서 RestArt 모바일 리플릿 기능을 실행한다.
2. 마음에 드는 작품을 촬영한다.
3. 전시 관람이 끝난 후 리플릿 생성을 선택한다.
4. 서비스는 촬영 이미지를 분석해 취향 결과를 생성한다.
5. 사용자는 개인화된 모바일 리플릿을 확인한다.
6. 리플릿을 SNS에 공유하거나 저장한다.

### 기대 효과

- 전시 경험의 디지털 기록
- 개인 취향 분석 제공
- 전시 관람 후에도 지속되는 콘텐츠 경험
- SNS 공유를 통한 전시 홍보 효과

---

## Scenario 4. 작품을 등록하고 싶은 작가

### 사용자 상황

작가는 자신의 작품을 더 많은 사람에게 알리고 싶지만 독자적인 홍보 채널과 판매 구조가 부족하다.

특히 신진 작가와 장애 예술인은 기존 갤러리, 전시, 플랫폼에 접근하기 어렵다.

### 사용 흐름

1. 작가는 작가 페이지를 생성한다.
2. 작품 이미지, 제목, 설명을 등록한다.
3. 서비스는 작품의 색상과 감성을 자동 추출한다.
4. 작가는 AI가 추출한 감성 정보를 검수하고 수정한다.
5. 작품은 RestArt 작품 DB에 등록된다.
6. 사용자의 공간과 취향에 따라 작품이 추천된다.
7. 작품 구매 또는 리세일 흐름으로 연결된다.

### 기대 효과

- 작품 노출 기회 확대
- 작가의 의도 반영 가능
- 추천 시스템을 통한 새로운 고객 접점 확보
- 작품 판매 가능성 증가

---

## Scenario 5. 전시 홍보가 필요한 소규모 전시 운영자

### 사용자 상황

소규모 전시 운영자는 자체 앱이나 디지털 홍보 시스템을 구축하기 어렵다.

전시 홍보를 위해 SNS를 활용하고 싶지만, 관람객이 자발적으로 공유할 수 있는 콘텐츠를 제작하기 쉽지 않다.

### 사용 흐름

1. 전시 운영자는 전시 정보와 작품 정보를 등록한다.
2. 관람객은 전시장에서 작품을 촬영한다.
3. 관람객별 개인화 리플릿이 생성된다.
4. 리플릿은 SNS 공유에 최적화된 형태로 제공된다.
5. 관람객이 공유한 리플릿을 통해 전시 홍보가 확산된다.

### 기대 효과

- 별도 인프라 없이 디지털 홍보 가능
- 관람객 참여형 콘텐츠 확보
- 전시 인지도 상승
- 전시 종료 후에도 콘텐츠 확산 가능

---

## Scenario 6. 작품 컬렉션을 관리하고 싶은 사용자

### 사용자 상황

사용자는 구매한 작품 또는 관심 작품을 한곳에 모아 관리하고 싶다.

작품을 단순히 구매하는 데서 끝내지 않고, 자신의 취향과 컬렉션을 기록하고 공유하고 싶다.

### 사용 흐름

1. 사용자는 추천 작품을 컬렉션에 저장한다.
2. 구매한 작품은 개인 컬렉션에 자동 등록된다.
3. 사용자는 자신의 컬렉션을 공개하거나 비공개로 설정한다.
4. 컬렉션을 기반으로 추가 작품을 추천받는다.
5. 필요 시 리세일 기능을 통해 작품을 재거래한다.

### 기대 효과

- 작품 소유 경험 강화
- 개인 취향 데이터 축적
- 리세일 기반 아트테크 경험 제공
- 사용자 커뮤니티 형성 가능

---

## Scenario 7. 기업 또는 기관이 작품을 렌탈하고 싶은 경우

### 사용자 상황

기업이나 기관은 사무실, 로비, 행사장, 공공공간에 어울리는 작품을 배치하고 싶다.

동시에 ESG, 문화예술 후원, 장애·신진 예술인 지원과 연결되는 활동을 원한다.

### 사용 흐름

1. 기업 또는 기관은 공간 정보와 목적을 등록한다.
2. RestArt는 공간 성격에 맞는 작품을 큐레이션한다.
3. 작품 렌탈 또는 정기 교체 구성을 제안한다.
4. 예술인의 작품이 실제 공간에 설치된다.
5. 기업 또는 기관은 문화예술 후원 및 ESG 활동으로 활용한다.

### 기대 효과

- 공간 분위기 개선
- 문화예술 후원 효과
- 장애·신진 예술인 지원
- B2B/B2G 수익 모델 확장

---

## 사용자 시나리오 요약

RestArt의 사용자 시나리오는 다음 세 가지 방향으로 정리된다.

1. 사용자는 자신의 공간과 취향에 맞는 작품을 더 쉽게 발견한다.
2. 예술인은 작품을 더 많은 사용자에게 노출하고 판매 기회를 얻는다.
3. 전시와 기관은 개인화된 디지털 콘텐츠를 통해 홍보와 참여를 확대한다.
