# Problem Definition

## 문제 정의

RestArt는 예술인과 소비자 사이에 존재하는 정보, 접근성, 신뢰, 경험의 단절을 해결하기 위해 기획되었다.

장애·신진 예술인은 작품을 지속적으로 생산하더라도 작품이 충분히 노출되지 못하고, 사용자는 자신에게 맞는 작품을 발견하거나 구매하기 어렵다. 또한 온라인 환경에서는 작품이 실제 공간에 어울릴지 판단하기 어렵기 때문에 구매 전환까지 이어지기 어렵다.

## 문제 1. 장애·신진 예술인의 낮은 작품 노출과 소득 문제

장애·신진 예술인은 지자체나 공공 지원이 존재하더라도 안정적인 작품 판매 구조를 확보하기 어렵다. 인지도 부족, 전시 기회 부족, 홍보 채널 부족, 작품 유통 구조의 한계로 인해 작품이 소비자에게 도달하지 못하는 경우가 많다.

이 문제는 단순히 작품 판매의 문제가 아니라 예술 활동의 지속 가능성과 연결된다. 작품이 시장에서 발견되지 못하면 예술인은 창작을 지속하기 어렵고, 소비자는 다양한 예술 작품을 접할 기회를 잃게 된다.

## 문제 2. 사용자의 작품 구매 진입장벽

일반 사용자는 작품을 구매하고 싶어도 다음과 같은 어려움을 겪는다.

- 어떤 작품이 내 공간에 어울리는지 판단하기 어려움
- 작품 가격과 투자 가치에 대한 정보 부족
- 온라인 이미지와 실제 설치 느낌의 차이
- 작품 구매 경험 부족
- 신진 작가나 장애 예술인 작품을 발견하기 어려움

기존 플랫폼은 작품 정보와 가격을 제공하는 데 집중하는 경우가 많지만, 사용자의 공간과 취향에 맞는 작품을 구체적으로 추천하거나 구매 전 설치 경험을 제공하는 데에는 한계가 있다.

## 문제 3. 기존 아트테크 플랫폼의 한계

기존 아트테크 플랫폼은 경매, 조각투자, 작품 판매, 작가 정보 제공 중심으로 구성되는 경우가 많다. 이러한 구조는 작품을 이미 알고 있거나 투자 목적으로 접근하는 사용자에게는 유효하지만, 예술 소비 경험이 부족한 사용자가 처음 작품을 선택하는 과정에는 충분하지 않다.

RestArt는 사용자의 공간 이미지와 취향 데이터를 출발점으로 삼아 작품 추천을 제공한다. 이를 통해 단순히 작품을 나열하는 방식이 아니라 사용자의 생활 공간과 감성에 맞는 작품을 발견하는 구조를 만든다.

## 문제 4. 전시 경험의 일회성

전시 관람은 현장에서 강한 경험을 제공하지만, 관람 후에는 기록과 공유가 제한적이다. 기존 리플릿은 동일한 정보를 모든 관람객에게 제공하기 때문에 개인의 취향이나 실제 관람 경험을 충분히 반영하지 못한다.

관람객이 전시장에서 촬영한 작품, 관심을 보인 색상, 감성 정보를 바탕으로 개인화된 리플릿을 생성하면 전시 경험을 지속 가능한 디지털 콘텐츠로 확장할 수 있다.

## 문제 5. 소규모 전시의 디지털 홍보 한계

대형 미술관이나 기관은 자체 웹사이트, 앱, 디지털 미술관 기능을 제공할 수 있지만, 소규모 전시나 신진 작가 중심 전시는 별도 디지털 인프라를 구축하기 어렵다.

RestArt의 개인화 리플릿 기능은 전시와 작품 정보를 입력하면 별도 인프라 없이 디지털 홍보와 관람객 참여를 유도할 수 있는 구조를 제공한다.

## 해결 방향

RestArt는 위 문제를 다음 방식으로 해결한다.

1. 사용자 공간 기반 AI 작품 추천
2. 색상과 감성 데이터를 활용한 작품 매칭
3. AR 설치 시뮬레이션을 통한 구매 전 검증
4. 작품 거래와 리세일을 통한 예술인 수익 구조 형성
5. 개인화 모바일 리플릿을 통한 전시 경험의 디지털 확장
6. SNS 공유와 커뮤니티 기능을 통한 홍보 선순환 구조 구축
