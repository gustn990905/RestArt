# Project Overview

## 프로젝트명

RestArt

## 프로젝트 한 줄 소개

RestArt는 사용자의 공간과 취향을 분석하여 어울리는 예술 작품을 추천하고, AR 시뮬레이션과 작품 거래 기능으로 연결하는 AI·AR 기반 아트테크 플랫폼이다.

## 프로젝트 배경

RestArt는 장애·신진 예술인의 작품이 충분히 발견되지 못하고, 사용자는 자신에게 맞는 작품을 고르기 어렵다는 문제에서 출발했다.

기존 온라인 미술 플랫폼은 작품 판매, 단순 정보 제공, 조각투자, 경매 중심으로 운영되는 경우가 많다. RestArt는 이와 달리 사용자의 공간 이미지와 취향 데이터를 분석해 작품을 추천하고, AR을 통해 실제 공간에 설치했을 때의 느낌을 확인할 수 있도록 설계되었다.

## 서비스 정의

RestArt는 다음 세 가지 기능을 중심으로 구성된다.

1. AI 기반 작품 추천
   - 사용자가 업로드한 공간 이미지에서 색상 정보를 추출
   - 색상과 감성을 매핑하여 공간에 어울리는 작품 추천
   - 작품 DB의 색상, 감성, 설명 데이터를 함께 활용

2. AR 작품 설치 시뮬레이션
   - 추천된 작품을 사용자의 실제 공간에 가상 배치
   - 구매 전 크기, 분위기, 공간 적합성을 확인
   - 온라인 작품 구매의 불확실성 감소

3. 작품 거래 및 리세일 구조
   - 예술인의 작품 판매 기회 확대
   - 사용자의 작품 구매, 컬렉션, 재거래 가능성 제공
   - 작품을 단순 장식품이 아니라 가치 소비와 아트테크 대상으로 연결

## 확장 기능

RestArt는 전시 경험 기반의 개인화 모바일 리플릿 기능으로 확장될 수 있다.

사용자가 전시장에서 촬영한 작품 이미지를 바탕으로 개인 취향을 분석하고, 전시 테마와 사용자 취향을 반영한 모바일 리플릿을 자동 생성한다. 리플릿에는 대표 작품, 취향 분석 결과, 추천 작품, 전시 정보, SNS 공유 기능이 포함된다.

이 기능은 일회성으로 끝나는 전시 경험을 개인화된 디지털 콘텐츠로 전환하고, 전시 홍보와 관람객 재방문 유도에도 활용될 수 있다.

## 핵심 사용자

### 일반 사용자

- 자신의 공간에 어울리는 작품을 찾고 싶은 사용자
- 인테리어와 가치 소비에 관심 있는 사용자
- 작품 구매 전 실제 공간 배치 느낌을 확인하고 싶은 사용자
- 전시 경험을 기록하고 공유하고 싶은 사용자

### 예술인

- 작품 노출과 판매 기회가 부족한 장애·신진 예술인
- 기존 갤러리나 대형 플랫폼에 접근하기 어려운 작가
- 작품의 감성, 색상, 공간 적합성을 기반으로 새로운 고객과 연결되고 싶은 작가

### 전시 및 기관 운영자

- 전시 홍보를 디지털 방식으로 확장하고 싶은 전시 운영자
- 관람객 참여형 콘텐츠가 필요한 문화예술 기관
- 소규모 전시의 디지털 접근성을 높이고 싶은 기관
