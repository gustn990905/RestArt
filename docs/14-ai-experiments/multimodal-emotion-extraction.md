# Multimodal Emotion Extraction

이 문서는 RestArt prototype에서 진행한 LLaVA 기반 이미지 감성 추출 실험을 정리한다.

이 실험은 이미지와 텍스트 prompt를 함께 처리할 수 있는 multimodal model을 활용하여, 작품 이미지 또는 사용자 입력 이미지에서 감성 후보를 추론하고 이를 추천 로직에 연결할 수 있는지 검토하기 위한 실험이다.

## 실험 목적

RestArt의 작품 추천은 색상 정보와 감성 정보를 함께 활용한다.

색상 정보는 이미지의 시각적 특성을 설명하고, 감성 정보는 이미지가 전달하는 분위기와 정서적 인상을 설명한다.

LLaVA 기반 감성 추출 실험의 목적은 다음과 같다.

- 이미지에서 감성 또는 분위기 후보를 추론할 수 있는지 확인
- 작품 이미지와 감성 태그 사이의 연결 가능성 검토
- 공간 이미지의 분위기를 작품 추천 기준으로 사용할 수 있는지 검토
- 색상 기반 추천을 감성 기반 추천으로 보완
- 향후 multimodal recommendation pipeline의 실험 근거 확보

## 실험 배경

RestArt는 사용자가 입력한 공간 이미지, 전시 이미지, 작품 이미지에서 색상과 감성 정보를 추출하고, 이를 작품 추천에 활용하는 구조를 가진다.

기존 색상 기반 추천은 이미지의 대표 색상과 작품 색상 정보를 비교하는 방식이다.

그러나 작품 추천에서는 색상만으로 설명하기 어려운 분위기, 감정, 장면의 인상도 중요하다.

따라서 multimodal model을 활용하여 이미지의 감성적 특성을 추론하는 실험을 진행하였다.

## 실험 흐름

LLaVA 기반 이미지 감성 추출 실험은 다음 흐름으로 정리할 수 있다.

1. 이미지 입력
2. 감성 후보 또는 분위기 분류 prompt 구성
3. multimodal model에 이미지와 prompt 입력
4. 이미지와 가장 가까운 감성 또는 분위기 추론
5. 추론 결과를 감성 태그 또는 추천 기준으로 변환
6. 작품 추천, 공간 기반 작품 추천, 모바일 리플릿 추천과 연결 가능성 검토

## 1. 이미지 입력

실험의 입력은 작품 이미지 또는 사용자 이미지다.

이미지는 model이 시각적 정보를 이해하는 기준 데이터로 사용된다.

입력 이미지는 다음 기능과 연결될 수 있다.

- 작품 이미지 감성 분석
- 공간 이미지 분위기 분석
- 전시 이미지 또는 리플릿 이미지 분석
- 사용자 입력 이미지 기반 추천

## 2. 감성 후보 prompt 구성

감성 추출을 위해서는 model에 이미지와 함께 감성 후보 또는 분류 기준을 제공한다.

prompt는 이미지가 어떤 분위기와 가까운지 판단하도록 구성된다.

예를 들어 다음과 같은 방식의 질문 구조를 사용할 수 있다.

- 이 이미지는 어떤 감성에 가장 가까운가?
- 이 작품 이미지는 어떤 분위기를 가진다고 볼 수 있는가?
- 주어진 감성 후보 중 이미지와 가장 가까운 감성은 무엇인가?

## 3. multimodal model 입력

이미지와 prompt를 함께 multimodal model에 입력한다.

이 실험에서는 LLaVA 계열 model을 활용하여 이미지와 텍스트 정보를 함께 해석하는 방향을 검토하였다.

## 4. 감성 또는 분위기 추론

model은 이미지의 시각적 특징과 prompt를 바탕으로 감성 또는 분위기 후보를 추론한다.

이 결과는 작품 감성 태그, 사용자 이미지 감성, 공간 분위기 정보와 연결될 수 있다.

## 5. 추천 기준으로 변환

추론된 감성 결과는 추천 로직에서 사용할 수 있는 형태로 변환된다.

예를 들어 다음과 같은 구조로 연결할 수 있다.

- 이미지 감성 결과
- 작품 감성 태그
- 공간 분위기 정보
- 감성 기반 추천 기준
- 색상 기반 추천 결과를 보완하는 추가 기준

## RestArt 추천 구조와의 연결

이 실험은 RestArt의 다음 기능과 연결된다.

## 공간 기반 작품 추천

공간 기반 작품 추천에서는 사용자가 입력한 공간 이미지의 색상과 분위기를 분석한다.

LLaVA 기반 감성 추출은 공간 이미지가 주는 분위기를 감성 정보로 변환하고, 이 정보를 작품 감성 태그와 비교하는 데 활용될 수 있다.

이를 통해 단순히 색상이 비슷한 작품뿐 아니라, 공간의 분위기와 어울리는 작품 추천이 가능해진다.

## 모바일 리플릿 생성

모바일 리플릿 생성에서는 사용자가 입력한 전시장 또는 작품 이미지와 관련된 추천 결과를 구성한다.

감성 추출 결과는 리플릿 결과 안에서 감성 기반 추천 작품을 선정하는 기준으로 활용될 수 있다.

## 작품 감성 태그 생성

작품 이미지에 대한 감성 추출 결과는 작품 DB의 감성 태그를 보완하는 데 사용할 수 있다.

이를 통해 작품 설명이나 수동 태그에만 의존하지 않고, 이미지 자체에서 감성 정보를 추론하는 추천 구조로 확장할 수 있다.

## 색상 분석과의 관계

색상 분석과 감성 추출은 서로 다른 기준을 제공한다.

| 구분      | 색상 분석                     | 감성 추출                        |
| --------- | ----------------------------- | -------------------------------- |
| 주요 입력 | 이미지 픽셀, RGB 값           | 이미지, 텍스트 prompt            |
| 주요 기준 | 대표 색상, 색상 cluster       | 분위기, 감성 후보, 이미지 인상   |
| 주요 결과 | 색상명, RGB, 색상 그룹        | 감성 태그, 분위기 분류           |
| 추천 역할 | 시각적 유사성 판단            | 정서적 적합성 판단               |
| 연결 기능 | 색상 기반 추천, 리플릿 디자인 | 감성 기반 추천, 작품 분위기 설명 |

두 분석 결과를 함께 사용하면 색상 유사성과 분위기 적합성을 동시에 고려하는 추천 구조를 만들 수 있다.

## 실험 범위

이 실험은 production-ready AI service가 아니라, RestArt 추천 구조에서 multimodal emotion extraction을 사용할 수 있는지 검토한 prototype experiment다.

현재 repository에서는 실험 notebook과 요약 문서를 포함하되, 실행 환경에서 자동 생성되는 cache, private dataset, model token, API key 등은 포함하지 않는다.

## 포함하지 않는 항목

이 repository에는 다음 항목을 포함하지 않는다.

- 비공개 API key
- model access token
- private dataset
- 민감한 원본 이미지 데이터
- 내부 기획 문서 원본
- cache 또는 output 파일
- 실행 환경에 따라 자동 생성되는 임시 파일

## 향후 확장 가능성

이 실험은 다음 방향으로 확장될 수 있다.

- 작품 이미지별 감성 태그 자동 생성
- 공간 이미지의 분위기 자동 분석
- 색상 기반 추천과 감성 기반 추천의 가중치 결합
- LLM 기반 추천 결과 설명 문장 생성
- 모바일 리플릿 결과의 감성 설명 자동 생성
- AR 작품 배치 결과에 대한 분위기 설명 생성
