# Recommendation Data Flow

## 개요

이 문서는 RestArt의 추천 데이터 흐름을 정리한다.

RestArt의 추천 시스템은 하나의 작품 DB를 중심으로 두 가지 사용자 경험을 지원한다.

1. 공간 이미지 기반 작품 추천
2. 전시 경험 기반 모바일 리플릿 추천

두 기능은 별도 프로젝트가 아니라, 작품 DB, 색상 테이블, 감성 테이블, 추천 알고리즘을 공유하는 RestArt 플랫폼 내부 기능이다.

---

## 1. 작품 DB 생성 흐름

작품 DB는 추천 시스템의 기반 데이터이다.

### 입력 데이터

작가가 입력하거나 시스템이 추출하는 데이터는 다음과 같다.

- 작가 닉네임
- 작품명
- 작품 설명
- 작품 이미지
- 감성 태그
- RGB 색상값
- 색상 비율
- 대표 색상

---

## 2. 작품 등록 및 분석 과정

작품 등록 흐름은 다음과 같다.

```text
작가 작품 등록
→ 작품 이미지 업로드
→ 작품명 입력
→ 작품 설명 입력
→ 감성 추출 알고리즘 실행
→ RGB 추출 알고리즘 실행
→ 작가 감성 검수 및 수정
→ 작품 DB 저장
```
