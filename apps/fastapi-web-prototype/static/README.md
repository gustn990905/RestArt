# Static Assets

이 폴더는 FastAPI web prototype의 static asset 위치를 유지하기 위한 공간이다.

원본 prototype에는 화면별 이미지, 폰트, 외부 JavaScript/CSS library, 작품 이미지가 함께 포함되어 있었지만, 이 repository에서는 코드 구조 확인에 필요한 핵심 source 중심으로 정리한다.

따라서 다음 항목은 포함하지 않는다.

- 작품 이미지
- 일반 이미지 asset
- font file
- 외부 library bundle
- upload 결과물
- clustering 결과물

실제 실행 환경에서는 template에서 참조하는 CSS, JavaScript, image asset을 별도로 배치해야 한다.
